# rgstephens Homebrew Tap

Homebrew tap for shipping my CLI tools

## Install

```bash
brew tap rgstephens/tap
```

## Install Packages

```bash
# formula
brew install rgstephens/tap/<name>
```

## Packages

### Formulae

- `fargo` — Fargo pool handicap tool
- `ghin` — Golf Handicap Information Network tool
- `poeusage` - [Poe usage tool](https://github.com/rgstephens/poeusage)

## Update / Uninstall

```bash
brew update

brew uninstall <formula>
```
